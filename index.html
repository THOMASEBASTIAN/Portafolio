¡Hola! Soy Thomas Bastos

Soy un desarrollador de software apasionado por la tecnología y la resolución creativa de problemas.

Mi foco principal es el desarrollo web, utilizando tecnologías como HTML, CSS y JavaScript para crear interfaces dinámicas y funcionales. Para asegurar un código limpio y un trabajo colaborativo eficiente, utilizo el control de versiones con GitHub. Actualmente, estoy expandiendo mis habilidades y aprendiendo sobre bases de datos para construir aplicaciones aún más robustas y completas.

Me considero una persona con habilidades blandas muy sólidas. Disfruto enfrentando desafíos y siempre busco la solución más eficiente para cualquier problema. Mi curiosidad no se detiene en el código; también soy un gran aficionado a los videojuegos y a la música, pasiones que inspiran mi creatividad.
